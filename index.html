About

This is an experiment in using the Web Audio API for sound analysis and visualisation byKatrina Hayes- a web developer in Sydney.

The visualisation works by continually measuring the frequency of Elliot Smith's song, Waltz #2, and uses this data to render a waveform, a frequency bar chart and a pulsing square.
